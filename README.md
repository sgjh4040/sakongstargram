## sakongstargram
40Stagram API server

## 소개
Instagram을 클로닝 한 40Stagram 에 사용되는 server 입니다.
데이터베이스 tool로 Prisma를 사용 하였습니다.
Query Language 중 하나인 graphql 를 기반으로 만들어 졌습니다.
실시간 채팅기능을 위한 subscription 기능도 포함되어 있습니다.
